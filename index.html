Mon thème

Définition du RGPD

Règlement Général sur la Protection des Données
Le RGPD encardre le traitementdes données personnelles sur le territoire de l'UE
Données personnelles : Toute information se rapportant à une personne physique indentifiée ou identifiable
Entrée en vigueur le 25mai 2018

Principe

Respecter la collecte / le traitement / la conservation des données
Permettre à la personne concernée de pouvoir : visualiser / rectifier / supprimer ses données personnelles

Pourquoi ?

Car une entreprise peut être amenée à recolter ses information (lorsqu'un client par exemple : se connecte sur leur site / ou dans les magasins à remplir un questionnaire afin de bénéficier d'une carte de fidélité / ou pour remplir leur base de données)
Ces données peuvent servir : à personnaliser les sites ou offres... des entreprises / réaliser des études / permettre les inscriptions de nouveau client ou salarié...

Risques

Perte de l'image de l'entreprise
Risque de perte de client / chiffre d'affaires
Problème avec la justice
Frais important (avocats / amandes...)
